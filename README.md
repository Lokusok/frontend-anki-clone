## Frontend для Anki-клона

Приложение для эффективного запоминания.

- Vue3
- Composition API
- TypeScript
- Pinia
- Axios + Interceptors (для токенов Laravel Sanctum)
- Vuetify
- Регистрация, авторизация
- Сброс пароля

---

## Запуск проекта

1. `docker compose up --build`
2. `curl -XGET http://localhost:5173`
